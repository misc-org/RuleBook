## 環境

ノート PC は、Windows 11 がインストールされています。基本的な操作方法は、Windows 11 の使い方を参照してください。

- [Windows 11 の使い方](https://www.microsoft.com/ja-jp/windows/windows-11)
- [Windows 11 の使い方（Microsoft 公式）](https://support.microsoft.com/ja-jp/windows/windows-11-help)

## 周辺

四つ並べている机の中心からコードが伸びています。ノート PC ですので、コードが抜けていたりコンセントに刺さっていないときは、必ずコードを差し込んでください。

使用後は、必ずコードや机の整理・整頓をしてください。

## 電源管理

基本的に、使い終わったらログアウトをしてシャットダウンです。スリープや休止状態にしないでください。

- キーボード右上の電源ボタンを押して電源を入れてください
- 使い終わったらスタートメニューからシャットダウンを選択してください
- 画面やボタンのライトが真っ暗になるまで待ってから、パソコンを閉じてください
- 諸事情により一時的に離席する場合や、更新等により終了が困難な場合は、近くの部員にお願いして、代わりにシャットダウンしてもらってください

## ログイン

アカウントは、入学後に配布されるものを使用してください。アカウント情報は他の部員に教えないでください。

情報システム部部員として、まさかそんなことはないと思いますが、もしパスワードを忘れた場合は、顧問に相談してください。

## ファイル管理

使用者は自分自身のみではないので、必要以上のファイルを保存しないでください。また、他の部員のファイルを削除しないでください。

基本的に、一度保存して、使い終わったら削除してください。

プロジェクト等で必要なファイルは、USB メモリ等に保存して持ち帰るか、クラウドサービスに保存してください。
MISC では GitHub を推奨しています。使い方等、詳しく知りたい場合は先輩に聞くか、[GitHub の公式サイト](https://github.com/) を参照してください。

### N ドライブ

ノート PC には、N ドライブが割り当てられています。N ドライブには、ノート PC で共有される `report-T` フォルダと、自分専用の `report-学年-学籍番号` フォルダがあります。

`report-T` フォルダは、顧問など一部の人しかファイルの追加等の作業ができません。通常は閲覧のみです。 `report-学年-学籍番号` フォルダは、自分専用のフォルダです。他の部員はアクセスできません。

## インターネット

インターネットは、学校側が管理しているため、http 通信等、一部の通信が制限される場合があります。そのため、一部のサイトにアクセスできない場合があります。

また、自身のスマートフォン等を使って、インターネットに接続することは基本的に禁止です。
必要があれば、顧問や部長に確認を取ってください。

## ソフトウェア

ノート PC には、基本的なソフトウェアがインストールされています。
通常のアカウントではインストール等を制限していますので、必要なソフトウェアがあれば、顧問や部長に相談するか、[共用 PC](./common-pc) を使用してください。

### インストール済みのソフトウェア

- Visual Studio Code
- Google Chrome

## その他

あくまでも、ノート PC は部活動用です。個人の使用は禁止です。

個人の範疇を超える使用・過度な負荷がかかるような操作を行った等の目撃・報告があった場合、部長や顧問と相談し、対処を行う場合があります。
